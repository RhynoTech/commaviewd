# Changelog

## [0.0.52-alpha] - 2026-06-01

### Improved

- Made the CommaView runtime integration with openpilot/sunnypilot more resilient to upstream UI changes. The installer now updates known safe hook points instead of relying on a brittle static patch.
- Improved install, upgrade, repair, and uninstall safety with better backups, cleaner rollback behavior, and recovery backups that survive failed installs or removals.
- Improved repair behavior when upstream UI files have drifted or been partially modified: CommaView now fails closed, preserves recovery data, and avoids leaving partially-applied changes behind.
- Added support for newer onroad UI file layouts used by current openpilot/sunnypilot branches.
- Improved runtime UI reload handling so CommaView changes are loaded when safe, and deferred when the device is onroad.
- Exported sunnypilot speed-limit pre-active arrow state so CommaView can match the upstream MICI `+` / `-` speed-limit prompt.

### Fixed

- Fixed cases where a failed install or repair could leave managed openpilot/sunnypilot UI files in a partially changed state.
- Fixed uninstall ordering so CommaView’s UI integration is reverted before runtime services and files are removed.
- Fixed verification so all supported onroad UI hook locations are checked, not just the first matching path.
- Fixed release publication safeguards so release assets are validated before publishing and “current release” promotion is an explicit step.

### Validation

- Added stronger release, CI, and device-test coverage for runtime install/repair/uninstall behavior.
- Added release provenance artifacts for easier troubleshooting and auditability.
- Expanded upstream canary coverage to catch openpilot/sunnypilot UI drift earlier.

### Notes

- No intentional video streaming or Android app transport changes from `v0.0.51-alpha`.
- This release adds sunnypilot-only speed-limit pre-active telemetry fields for MICI HUD parity.
- CommaView runtime support remains intentionally limited to official `commaai/openpilot` and known sunnypilot remotes.
