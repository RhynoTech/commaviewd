# Changelog

## [0.0.54-alpha] - 2026-06-10

### What changed

- This runtime is built for CommaView Android app `0.0.157-alpha`.
- CommaView now repairs/applies the right UI export patch for supported OpenPilot and SunnyPilot installs instead of relying on one brittle prebuilt patch.
- Latest OpenPilot, latest SunnyPilot, and SunnyPilot users still on the May 27 release tag (`v2026.001.007`) are covered by this release.
- If OpenPilot or SunnyPilot changes after CommaView is installed, the runtime now detects that and asks you to repair/reinstall the patch.

### Why you should update

- Fixes broken or missing overlays caused by recent OpenPilot/SunnyPilot UI and schema changes.
- Makes repair safer: the runtime records the exact upstream commit it patched and refuses to pretend an old patch is still valid after upstream updates.
- Prevents one bad telemetry/service payload from stopping later overlay data such as model path, calibration, car output, or onroad projection.

### Compatibility

- Compatible Android app: `0.0.157-alpha` / tag `app-v0.0.157-alpha`.
- Supported upstreams: official `commaai/openpilot` and known SunnyPilot remotes.
- Verified before release on current OpenPilot refs, current SunnyPilot refs, SunnyPilot `master`, and SunnyPilot `v2026.001.007`.
- Older or custom forks are handled safely: if the local source still matches what CommaView can transform, repair should work; if not, CommaView should fail safe and ask for a newer runtime instead of half-patching.

### After installing

- Run CommaView repair/reinstall after updating OpenPilot or SunnyPilot.
- If status says repair is needed, that usually means the upstream checkout changed since CommaView last patched it.
- First-drive/runtime status may still say telemetry is not proven until real data is seen; that is expected for this alpha.

### Fixed

- Fixed SunnyPilot overlay starvation caused by `DriverMonitoringState` schema drift.
- Fixed stale patch detection when OpenPilot/SunnyPilot updates under an existing CommaView install.
- Removed unsupported comma 3/TICI branches from release blockers; CommaView support remains scoped to MICI/comma 4.

### Testing focus

- Fresh install and repair on latest OpenPilot/SunnyPilot.
- SunnyPilot May 27 users (`v2026.001.007`).
- Road path, model lines, torque arc, confidence ball, driver monitoring, camera switching, projection alignment, and telemetry freshness after repair.
