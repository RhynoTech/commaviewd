# Changelog

## [0.0.51-alpha] - 2026-05-17

### Fixed

- Local device discovery now includes the runtime version under both `version` and `runtimeVersion`, matching the control API and Android app expectations.
- Improved compatibility with newer CommaView Android builds when they identify discovered devices and display installed runtime version details.

### Changed

- Expanded release validation around pairing, local discovery, and raw telemetry envelope contracts.

### Notes

- No video streaming or telemetry payload behavior changed in this release.
- This release is intended as a small compatibility/runtime-validation release for the next Android app alpha.
